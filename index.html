Front - End Vs Back - End

Front - End

Bagian front-end adalah bagian yang dapat langsung dilihat dan diinteraksikan oleh user. Bagian ini dibangun menggunakan HTML, CSS, dan JavaScript. HTML adalah tulang punggung dari website. HTML menentukan struktur dan konten website. Versi terakhir dari HTML adalah HTML5. CSS adalah bahasa pemograman yang mengontrol tampilan HTML pada halaman website. CSS menentukan warna, font, gambar background, dan bahkan bagaimana tampilan keseluruhan website itu sendiri. Versi terakhir dari CSS adalah CSS3, dimana sudah terdapat fitur seperti interaktifitas dan animasi dasar. Kita bisa menciptakan sebuah website hanya dengan menggunakan HTML dan CSS, tetapi kita dapat membuat website lebih menarik dengan JavaScript. Dengan JavaScript, kita bisa melakukan banyak hal, seperti interaktifitas, animasi yang lebih kompleks, dan bahkan bisa membuat fully feature web application.


Back - End

Back-end adalah bagian belakang layar dari sebuah website. Bahasa pemograman untuk back-end development diantaranya adalah PHP, Ruby, Python, dan banyak lainnya. Satu hal yang biasa terjadi: tidak akan banyak perusahaan yang mem-posting lowongan kerja dengan kalimat “back-end developer”. Yang akan lebih sering ditemui adalah lowongan pekerjaan untuk “Ruby developer”, “PHP developer”, dll.
